# Display Launcher

Экспериментальная material-версия на основе рабочей безопасной сборки.

Что внутри:
- список всех установленных пользовательских приложений с иконкой запуска
- исключены системные приложения
- 4 режима запуска: Главный, Пассажирский, Весь экран, Потолочный
- кнопка «Создать ярлыки» для выбранного приложения
- для Яндекс Навигатора сохранён явный запуск через NavigatorActivity
- для остальных приложений используется launcher activity

Важно:
- список строится по приложениям с обычным ярлыком запуска
- на некоторых устройствах запуск на нужный displayId зависит от политики прошивки
- ярлыки работают через ShortcutProxyActivity


Ярлыки теперь создаются по одному: после нажатия «Добавить» для текущего ярлыка появится следующий.


Изменения: иконка обновлена под широкую рамку, создание ярлыков переведено на legacy INSTALL_SHORTCUT.
