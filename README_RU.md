# Display Launcher

Экспериментальная material-версия на основе рабочей безопасной сборки.

Что внутри:
- список установленных пользовательских приложений (с отбором в настройках)
- исключены системные и вендорские приложения
- 4 режима запуска: Основной, Пассажирский, Весь экран, Потолочный
- быстрые кнопки запуска прямо в строке приложения
- долгое нажатие по кнопке экрана создаёт/снимает закреплённый ярлык
- вкладка Отладка с диагностикой и инженерными инструментами
- блок APK загрузка/установка через публичную папку Яндекс.Диска
- для Яндекс Навигатора сохранён явный запуск через NavigatorActivity
- для остальных приложений используется launcher activity

Важно:
- список строится по приложениям с обычным ярлыком запуска
- на некоторых устройствах запуск на нужный displayId зависит от политики прошивки
- закреплённые ярлыки создаются через alias-activity (самый стабильный способ для Flyme/Geely)
- pinned/legacy shortcuts используются как fallback, если alias недоступен


Закрепление: долгое нажатие на иконку экрана создаёт один ярлык на выбранный экран.
Повторное долгое нажатие снимает закрепление.

Отладка:
- переход через кнопку под блоком 2ГИС CARAPP
- диагностика дисплеев, закреплений и статуса ярлыков
- APK загрузка: список берётся из публичной папки Яндекс.Диска
- после скачивания кнопка меняется на «Установить» и запускается системный установщик
- долгий тап по кнопке обновления в шапке Отладки перезагружает список APK

Настройка источника APK:
- ссылка хранится в `app/src/main/res/values/strings.xml` (`debug_apk_public_url`)
